# WUIForm - WUI inputs demo

## Description

This demo shows the use of WUIForm with fill style and WUI inputs components.

## WUI JS Stack

- WUIIcon - 0.3
- WUIForm - 0.5
- WUISelectpicker - 0.5
- WUIDatepicker - 0.5
- WUITimepicker - 0.5
- WUIColorpicker - 0.5
- WUISwitch - 0.6
- WUIIntensity - 0.4
- WUIButton - 0.5
