# WUITable - WUI Paginator demo

## Description

This demo shows the use of WUITable's pagination functionality through the WUIPaginator component.

Table data is loaded directly in component creation.

## WUIJS Stack

- WUIPaginator - 0.1
- WUITable - 0.3
